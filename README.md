GEPROEST

Generador de Problemas Estadísticos

GEPROEST es una aplicación de escritorio que genera automáticamente múltiples versiones de un mismo problema de estadística con datos distintos, manteniendo la misma estructura.
Su objetivo es facilitar la evaluación académica, reducir el copiado entre estudiantes y ahorrar tiempo al docente.

![Uso de geoprest](https://github.com/user-attachments/assets/03cf582b-8b9d-4dbd-9dcd-efffe247f961)


Características principales

- Generación automática de versiones de un mismo problema estadístico
- Soporte para distintos módulos de estadística:
  - Estadística descriptiva
  - Diagramas de Venn
  - Tablas de contingencia
  - Distribución normal
  - Distribución binomial
- Exportación compatible con Moodle en formato XML
- Inclusión de datos en CSV embebidos en base64
- Configuración flexible del número de versiones generadas

Tecnologías utilizadas

C#
- .NET
- Visual Studio
- Windows Forms
- XML (formato de preguntas Moodle)
- CSV (almacenamiento de versiones de datos)

Instalación

Para probar el proyecto en tu máquina:
1. Descarga el instalador generado desde Visual Studio.
2. Ejecuta el archivo .exe.
3. Sigue el asistente de instalación.
4. Una vez finalizado, abre GEPROEST desde el menú de inicio o el acceso directo.
No se requiere configuración adicional ni dependencias externas.

Uso

1. Selecciona el tipo de problema estadístico.
2. Configura los parámetros del ejercicio.
3. Indica el número de versiones a generar.
4. Exporta los resultados para su uso en Moodle o revisión local.
